Tela de Login

Especificações Visuais

  • Resolução Base: 1920x1080px (Desktop)
  • Fundo: Gradiente de Azul Confiança (#1A5F7A) para Azul Escuro (#0D3B4F)
  • Logo: 120px de altura, centralizado no topo

Componentes Principais

  • Container de Login: Card branco com sombra suave
  • Campos de Entrada: Email e Senha com validação em tempo real
  • Botão de Login: Azul Confiança (#1A5F7A)
  • Link "Esqueci minha senha": Texto discreto abaixo do formulário
  • Opção "Criar Conta": Botão secundário na parte inferior

Comportamentos

  • Validação em tempo real dos campos
  • Feedback visual para erros de autenticação
  • Opção de mostrar/ocultar senha
  • Redirecionamento para dashboard após login bem-sucedido

Fluxo de Autenticação

O sistema utiliza autenticação via Supabase com tokens JWT, permitindo login com email/senha e também opções de login social (Google, Apple). O fluxo inclui verificação de email para novos usuários e recuperação de senha segura.

Tela de Criação de Conta

  • Formulário com campos para Nome, Email, Telefone, CPF, Senha
  • Seleção de tipo de usuário (Proprietário ou Motorista)
  • Upload de foto de perfil (opcional)
  • Termos de uso e política de privacidade
  • Botão de cadastro com feedback visual durante o processo

Dashboard do Proprietário

Especificações Visuais

  • Layout com menu lateral fixo à esquerda
  • Área de conteúdo principal à direita
  • Barra superior com notificações e perfil
  • Cards de resumo com indicadores visuais

Componentes Principais

  • Cards de Resumo: Receita, Veículos Ativos, Vistorias Pendentes, Documentos a Vencer
  • Gráfico de Receita: Visualização mensal/anual
  • Lista de Veículos: Status, motorista atual, quilometragem
  • Alertas e Notificações: Prioridade visual por cores

Seções Principais

Resumo Financeiro
  • Receita total do mês atual
  • Comparativo com mês anterior
  • Gráfico de barras para visualização mensal
  • Filtros por período e por veículo
Visão da Frota
  • Total de veículos por status
  • Taxa de ocupação da frota
  • Veículos com manutenção próxima
  • Quilometragem média da frota
Alertas e Lembretes
  • Documentos próximos do vencimento
  • Vistorias pendentes de aprovação
  • Multas recentes
  • Pagamentos atrasados
Ações Rápidas
  • Adicionar novo veículo
  • Registrar novo motorista
  • Agendar vistoria
  • Gerar relatório

Dashboard do Motorista

Especificações Visuais

  • Layout com menu lateral simplificado
  • Foco em informações do veículo atual
  • Cards de resumo com indicadores visuais
  • Acesso rápido a funções essenciais

Componentes Principais

  • Seção do Veículo Atual: Foto, placa, modelo, status
  • Cards de Resumo: Quilometragem semanal, vistorias pendentes, multas, documentos
  • Gráfico de Quilometragem: Visualização diária/semanal
  • Alertas e Lembretes: Próximas vistorias, documentos a vencer

Modais e Interações

Modal: Registrar Quilometragem
  • Campo para data do registro
  • Campo para quilometragem atual
  • Seletor de tipo de uso
  • Upload de foto do hodômetro
  • Campo para observações
Modal: Solicitar Vistoria
  • Seleção de tipo de vistoria
  • Seleção de data e horário preferencial
  • Seleção de local
  • Campo para motivo (se emergencial)
  • Campo para descrição do problema
Modal: Reportar Problema
  • Seleção de categoria do problema
  • Seleção de nível de urgência
  • Campo para descrição detalhada
  • Upload de fotos do problema
  • Seleção de impacto na utilização
Seção de Últimas Atividades
  • Timeline com histórico de ações
  • Registros de quilometragem
  • Vistorias realizadas
  • Multas recebidas
  • Pagamentos efetuados

Módulo de Veículos

Especificações Visuais

  • Visualização em lista e cards
  • Filtros avançados no topo
  • Indicadores visuais de status
  • Ações rápidas por veículo

Componentes Principais

  • Barra de Filtros: Status, motorista, marca, modelo
  • Visualização em Tabela: Dados técnicos e status
  • Visualização em Cards: Foto e informações principais
  • Botões de Ação: Editar, Atribuir Motorista, Agendar Vistoria

Detalhes do Veículo

Informações Gerais
  • Dados de Identificação: Placa, Renavam, Chassi
  • Dados Técnicos: Marca, Modelo, Ano, Cor
  • Status Atual: Ativo, Em Manutenção, Disponível
  • Motorista Atual: Nome, contato, desde quando
  • Dados Financeiros: Valor de aquisição, valor de locação
Abas de Detalhes
  • Histórico de Quilometragem: Gráfico e registros
  • Vistorias: Histórico e agendamentos
  • Documentos: CRLV, Seguro, Revisões
  • Multas: Histórico e status
  • Manutenções: Preventivas e corretivas
  • Histórico de Motoristas: Quem já utilizou o veículo

Formulário de Cadastro/Edição

Dados Básicos
  • Placa, Renavam, Chassi
  • Marca, Modelo, Versão
  • Ano de Fabricação, Ano Modelo
  • Cor, Combustível
  • Quilometragem Inicial
Dados Complementares
  • Data de Aquisição
  • Valor de Aquisição
  • Valor de Locação
  • Upload de Fotos (múltiplas)
  • Upload de Documentos
